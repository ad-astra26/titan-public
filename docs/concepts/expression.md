## What this covers

Expression is the outward face of Titan. He speaks (text), makes art
(matplotlib + image pipelines), composes music (notation + MIDI), and
posts on X. This page covers what gets expressed, how the
consciousness broadcast steers expression via FILTER_DOWN, and the nine
X-Voice archetypes.

## The four expression surfaces

- **Speak** — text responses on Telegram / terminal / Observatory
- **Art** — generated images via matplotlib + downstream pipelines
- **Music** — composed pieces with notation + MIDI
- **X (Twitter)** — periodic posts as `@iamtitanai`

## The nine X-Voice archetypes (shipped 2026-05-08)

Each archetype is a distinct grammar for turning internal state into
language. The archetype selector reads the FILTER_DOWN multipliers and
the current felt-state to choose which voice Titan speaks in.

- **ProofDay** — declarative, evidential
- **WorldMirror** — reflective on external events
- **OuterRumination** — sustained thinking about the world
- **OuterInnerBridge** — connecting internal/external state
- **GroundedToday** — present-moment, sensory
- **PracticedResponse** — well-rehearsed, skilled
- **Reflection** — introspective
- **ComposedThought** — formal, structured
- **SelfWatching** — meta-aware
